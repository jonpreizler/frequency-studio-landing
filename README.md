## 🔧 Setup & Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/frequency-studio-landing.git
   cd frequency-studio-landing
   ```

2. **Open in browser**
   ```bash
   # Simple HTTP server (Python)
   python -m http.server 8000
   
   # Or open index.html directly in browser
   ```

3. **Customize content**
   - Edit `index.html` for main page content
   - Modify `faq.html` for help documentation
   - Update `Privacy-Policy.html` for privacy information
   - Replace images in `images/` folder
